# 🍽️ NutriPlan+

NutriPlan+ is a web-based meal ordering system developed as a Diploma in Computer Science project. The system allows users to browse meals, manage their cart, and place orders, while administrators can manage menu items and monitor orders.

## 📌 Features

### Customer
- Browse available meals
- View meal details
- Add meals to cart
- Update cart quantity
- View order status
- Update Personal Information
- Update Password & Security
- Update Address

### Administrator
- Admin dashboard
- Add new menu items
- Update existing menu items
- Delete menu items
- View order history

## 🛠️ Technologies Used

- PHP
- HTML5
- CSS3
- MySQL
- XAMPP
